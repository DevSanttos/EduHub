# EduHub
EduHub é um sistema web desenvolvido com HTML, CSS e JavaScript, projetado para oferecer uma experiência simples, intuitiva e eficiente de aprendizagem online. Inspirado em plataformas modernas como Alura, Udemy e Coursera, o EduHub permite que usuários explorem cursos, assistam aulas, acompanhem seu progresso e gerenciem seus estudos de forma organizada.
